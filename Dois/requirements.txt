# Instruções de Instalação e Execução

## 1. Instalar as dependências:
pip install streamlit pandas plotly

## 2. Executar a aplicação:
streamlit run dna_streamlit_app.py

## 3. A aplicação abrirá automaticamente no navegador em:
http://localhost:8501

## Recursos da Aplicação Streamlit:

### Principais Melhorias em relação ao Tkinter:
1. **Interface Web Moderna**: Design responsivo e profissional
2. **Visualizações Interativas**: Gráficos com Plotly
3. **Navegação por Abas**: Organização intuitiva
4. **Métricas Visuais**: Cards com indicadores
5. **Filtros Avançados**: Busca e comparação
6. **Export de Dados**: Download em CSV
7. **Responsividade**: Adapta-se a diferentes telas

### Funcionalidades Exclusivas do Streamlit:
- **Session State**: Mantém dados entre interações
- **Caching**: Performance otimizada
- **Deploy Fácil**: Streamlit Cloud, Heroku
